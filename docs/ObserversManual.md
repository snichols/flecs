# Observers

<!-- TODO: port from /work/agents/claude/projects/SanderMertens/flecs/docs/ObserversManual.md (Phase 14.7) -->

This manual covers the observer API: `Observe`, `ObserveID`, `Observe2`, built-in events (`EventOnAdd`, `EventOnSet`, `EventOnRemove`), multi-subscriber patterns, and `Unsubscribe`.

Also covers component hooks (`OnAdd`, `OnSet`, `OnRemove`) and how they differ from observers (one hook per type per event vs. unlimited observers).

See the [Quickstart](Quickstart.md) for a hands-on introduction.
